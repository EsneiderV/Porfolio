MIS SERVICIOS

Tecnología Front-End

  • HTML
  • CSS
  • JavaScript

El Front-end es enfocado en el usuario, en todo con lo que puede interactuar y lo que ve mientras navega. Brindandole asi una buena experiencia de usuario, inmersión y usabilidad son algunos de los objetivos que busco, y hoy en día hay una gran variedad de frameworks, preprocesadores y librerías que me ayudan en esta tarea.

Tecnología Back-End

  • PHP
  • Python
  • JavaScript

El Back-End hace referencia al proceso del servidor y más recientemente procesos en la nube del sitio web. Encargado asi del sistema que recoge y guarda información y la organizar. Es decir, todo aquello con lo que el usuario no interactúa directamente; por ejemplo, las APIs o paquetes de aplicaciones, las librerías y componentes del sistema.

Base de datos

  • MySQL
  • MongoDB

En el desarrollo de aplicaciones de software una de las piezas fundamentales son las bases de datos, este siendo el modelo más habitual para el almacenamiento, con persistencia, de la información que manejan las aplicaciones. Los sistemas gestores de bases de datos que he utilizado son MySQL y MongoDB, siendo bases de datos relacionales y no-relacionales.
